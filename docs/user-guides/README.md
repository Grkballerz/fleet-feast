## Using These Guides

### For Customers
Start with the **Customer Guide** to learn how to:
1. Create your account
2. Find the perfect food truck
3. Book and pay for your event
4. Communicate with vendors
5. Manage your bookings

**Quick Start:** Jump to "Getting Started" → "Creating Your Account"

### For Vendors
Start with the **Vendor Guide** to learn how to:
1. Apply and get approved
2. Set up your profile
3. Accept booking requests
4. Get paid through escrow
5. Build your reputation

**Quick Start:** Jump to "Getting Started" → "Application Process"

### For Admins
Start with the **Admin Guide** to learn how to:
1. Navigate the admin dashboard
2. Review and approve vendors
3. Resolve disputes fairly
4. Manage violations
5. Monitor platform health

**Quick Start:** Jump to "Admin Dashboard Overview"
